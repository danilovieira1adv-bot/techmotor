## Packages
framer-motion | Smooth animations for page transitions and UI elements
recharts | Dashboard charts for metrics
lucide-react | Icons (already in base, but emphasizing usage)
clsx | Utility for conditional classes (standard)
tailwind-merge | Utility for merging tailwind classes (standard)
date-fns | Date formatting

## Notes
- Authentication is handled via Replit Auth (/api/login, /api/logout, /api/auth/user)
- WhatsApp simulation uses the Chat integration (/api/conversations) with SSE streaming
- AI Analysis uses /api/inspections/analyze
- Dark mode support via Tailwind 'dark' class
